Question-1: What are the different ways to select an element in the DOM?

We can select or catch a element from the dom using GetElementById("#id"), GetElementByTagName("p"), GetElementsByClass("className"), querySelector(selectors), querySelectorAll(".className")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The main differences are, we can get
innerHTML: HTML markup (tags and text).
innerText: Visible text (rendered).
textContent: All text (raw), and change as we wish.

Question-3: What is event delegation in the DOM?

We know that by default event bubble happens in our DOM. So we can use this for our benefit. Suppose we want to add Event Listener on every child node of a parent node, so instead of giving event listener on every child node we can simply put it on parent node thus it will save our storage and make our code faster.

Question-4: What is event bubbling in the DOM?

Suppose someone says you I am in my bedroom, now by this we can say that he is definately in his house also in his district also in his country and obviously in the world, right? Event bubbling is similar like this. Suppose in a HTML file there is a div inside another div if we tap on the inner div then logically tapping the outer div and also the body and the html and the document too. If there are multiple event listener on these elements than those will also hit. Some tme we need it and sometime we don't If we want to stop the propagation. we can use stopPropagation() thus only the targeted event will be triggered.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create any element by .createElement("div")
We can append it using .appendChild(div) , .append() , .prepend(), .insertBefore(newNode, referenceNode)
We can remove using .removeChild(), element.remove() etc.
